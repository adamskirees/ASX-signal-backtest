# ASX-signal-backtest

PROJECT: Algorithmic Backtester & Parameter Optimzer 
A Python-based backtesting engine designed to validate technical strategies on ASX-listed growth assets.

# Key Features:

Hyperparameter Optimization: Utilizes a Grid Search optimizer to identify the "Robust Zone" for moving average windows.

Case Study (XRO.AX): Optimized a 5/55 Weekly Golden Cross, achieving a 5.59x growth multiplier vs a lagging 3.7x with standard parameters.

Financial Rigour: Includes a comprehensive "Trade Log" generator for auditable performance tracking.

# Tech Stack: Python, Pandas, Matplotlib, YFinance.
